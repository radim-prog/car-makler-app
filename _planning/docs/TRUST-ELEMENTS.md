## 🧠 PSYCHOLOGY OF TRUST

### Trust Equation

```
Trust = (Credibility + Reliability + Intimacy) / Self-Interest

Where:
- Credibility: "Jsou experti?"
- Reliability: "Splní co slíbí?"
- Intimacy: "Cítím se s nimi v pohodě?"
- Self-Interest: "Nebo jen chtějí peníze?"

→ Naše úloha: Maximalizovat čitatel, minimalizovat jmenovatel
```

### Trust Building Timeline

```
0-5 sekund:   Visual trust (design, profesionalita)
5-30 sekund:  Social proof (reference, čísla)
30s-2 min:    Expertise signaling (jak to funguje)
2-5 min:      Deep dive (case studies, video)
5+ min:       Decision making (porovnání, cena)
```

**Každá fáze potřebuje své trust elementy.**

## 🚨 TRUST KILLERS (Avoid These!)

### ❌ Killer #1: Stock Photos

**Bad:**
```html
<img src="generic-handshake.jpg" />
```

**Why it kills trust:**
- Fake, inauthentic
- Everyone recognizes stock photos
- Signals "we don't have real content"

**Fix:**
- Use real photos (your team, your cars, your office)
- iPhone photos > fake stock photos
- Or hire photographer (1 day shoot = 100+ images)

### ❌ Killer #2: Typos & Grammar Errors

**Bad:**
```
Prodáme vaše auto rýchle a bez stresu.
```

**Why it kills trust:**
- Unprofessional
- "If they can't proofread, can they sell my car?"
- Small errors = big doubts

**Fix:**
- Proofread EVERYTHING
- Use Grammarly or similar tool
- Have native speaker review

### ❌ Killer #3: Fake Scarcity

**Bad:**
```
⏰ POUZE DNES! POSLEDNÍ ŠANCE!
[Shows same message every day]
```

**Why it kills trust:**
- User checks tomorrow, sees same message = liar
- Manipulative, not authentic
- Triggers spam detection

**Fix:**
- Use real scarcity (Kapacita: 12/15 aut tento měsíc)
- Or don't use scarcity at all
- Authenticity > tricks

### ❌ Killer #4: Overpromising

**Bad:**
```
ZARUČENĚ PRODÁME ZA 7 DNÍ!
NEJVYŠŠÍ CENA V ČR!
100% SPOKOJENOST!
```

**Why it kills trust:**
- Too good to be true = probably is
- Sets unrealistic expectations
- One broken promise = lost trust forever

**Fix:**
- Under-promise, over-deliver
- Use averages ("průměrně 17 dní" not "zaručeně 7")
- Show range ("94% spokojenost" not "100%")

### ❌ Killer #5: Hidden Costs

**Bad:**
```
Provize: 7 500 Kč*

[*plus administrativní poplatek 2 000 Kč,
marketing fee 1 500 Kč, success fee 3%, ...]
```

**Why it kills trust:**
- Bait and switch
- Feels manipulated
- Word of mouth spreads ("hidden fees!")

**Fix:**
- Total transparency
- Show all costs upfront
- If optional upsells, clearly mark as optional

## 🛡️ TRUST RECOVERY (When Things Go Wrong)

### Handling Negative Reviews

**Don't:**
- ❌ Delete/hide
- ❌ Argue with reviewer
- ❌ Ignore

**Do:**
- ✅ Respond publicly, professionally
- ✅ Acknowledge their concern
- ✅ Offer to resolve offline
- ✅ Show you care

**Template:**

```
Ahoj [Jméno],

Děkujeme za zpětnou vazbu. Mrzí nás, že vaše zkušenost
nebyla ideální.

Rádi bychom to napravili. Mohl byste nám zavolat na
773 838 773 nebo napsat na info@carmakler.cz?

Chceme se ujistit, že každý klient odchází spokojený.

S pozdravem,
[Jméno]
CAR makléř
```

**Other users see:**
- You respond to feedback (care)
- You try to resolve (proactive)
- You're reasonable (not defensive)

= Trust maintained (maybe even increased)

### Transparency in Mistakes

**If you mess up:**
- Admit it publicly
- Explain what happened
- What you're doing to fix it
- What you're doing to prevent future

**Example (blog post):**

```
# Co Se Pokazilo: BMW X3 a Naše Lekce

Minulý měsíc jsme udělali chybu. Prodávali jsme BMW X3
a podcenili jsme servisní historii. Auto mělo skrytou vadu,
která vyšla najevo 2 dny po prodeji.

Co jsme udělali:
1. Okamžitě kontaktovali kupce
2. Zaplatili opravu (5 000 Kč)
3. Omluvili jsme se
4. Vrátili část provize prodávajícímu

Co jsme se naučili:
- Důkladnější technická kontrola (nový protokol)
- Třetí strana check (externí mechanik)
- Buffer čas před prodejem

Chyby se stávají. Jak na ně reagujeme definuje, kdo jsme.

[Jméno], CEO
```

**Why this builds trust:**
- Honesty = rare = valuable
- Shows you learn
- Humanizes brand
- Accountability
